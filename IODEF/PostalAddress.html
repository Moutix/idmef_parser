PostalAddress

The PostalAddress class specifies a postal address formatted according to the POSTAL data type (Section 2.11).

Attributes

meaning (Optional)

A free-form description of the element content.

lang (Required)

A valid language code per RFC 4646 [7] constrained by the definition of "xs:language". The interpretation of this code is described in Section 6.
