Aggregates

NodeName (0..*)

The name of the Node (e.g., fully qualified domain name). This information MUST be provided if no Address information is given.

Address (0..*)

The hardware, network, or application address of the Node. If a NodeName is not provided, at least one Address MUST be specified.

Location (0..1)

A free-from description of the physical location of the equipment.

DateTime (0..1)

A timestamp of when the resolution between the name and address was performed. This information SHOULD be provided if both an Address and NodeName are specified.

NodeRole (0..*)

The intended purpose of the Node.

Counter (0..*)

A counter with which to summarizes properties of this host or network.
